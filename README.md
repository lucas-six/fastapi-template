# fastapi-template

FastAPI template. Using FastAPI, SQLModel, PostgreSQL, Docker, GitHub Actions, automatic HTTPS and more.

## Usage

### `.env`

```ini
# .env

APP_NAME="FastAPI App"
APP_VERSION="v1.0.0"
APP_DESCRIPTION="FastAPI app description."
DEBUG=true
```

### Run

```bash
uv run -m app.app
```
